# AWS Academy Cloud Foundations

Material del curso de AWS Academy Cloud Foundations - Fundamentos de AWS Cloud.

## Estructura del Curso

El curso está organizado en 8 módulos:

- [module-01](module-01/) - Introducción y navegación en la consola AWS
- [module-02](module-02/) - Redes Virtuales (VPC), subredes, routing y seguridad
- [module-03](module-03/) - IAM, CloudWatch y CloudTrail
- [module-04](module-04/) - Alta disponibilidad, ELB, Route 53 y caching
- [module-05](module-05/) - Facturación, Pricing y soporte AWS
- [module-06](module-06/) - Lambda y arquitecturas serverless
- [module-07](module-07/) - Containers (Docker, ECS, EKS)
- [module-08](module-08/) - Conexiones híbridas (Direct Connect, VPN, PrivateLink)

## Contenido de cada Módulo

Cada módulo contiene:

- `annexes/` - Material de referencia adicional
- `labs/` - Laboratorios prácticos con ejercicios

## Requisitos

- Cuenta AWS activa
- Navegador web moderno
