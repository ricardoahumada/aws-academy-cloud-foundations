# AWS Support Plans - Guía de Comparación

> **Tiempo de lectura:** ~8 min | **Referencia:** [AWS Support Plans](https://docs.aws.amazon.com/awssupport/latest/user/getting-started.html)

---

## Overview de Planes

AWS ofrece 4 planes de soporte que van desde gratuito hasta enterprise:

| Plan | Precio | Objetivo |
|------|--------|----------|
| **Basic** | Free | Learning, personal projects |
| **Developer** | $29/mes | Development, testing |
| **Business** | $100/mes | Production workloads |
| **Enterprise** | $15,000/mes | Mission-critical |

---

## Comparativa Detallada

### Basic Plan (Free)

**Incluye:**
- Documentación y whitepapers
- AWS re:Post community
- AWS Trusted Advisor (7 checks básicos)
- AWS Personal Health Dashboard
- 1 CloudWatch alarm/mes
- 20+ AWS tutorials y webinars

**Trusted Advisor Checks disponibles (Basic):**
```
✓ Security Groups - Ports unchecked
✓ S3 Bucket Permissions  
✓ RDS Snapshots
✓ IAM Use
✓ MFA on Root Account
✓ EBS Snapshots
✓ Auto Scaling Group configuration
```

**NO incluye:**
- Soporte técnico por email/chat/phone
- Soporte para arquitectura
- Training o certifications

---

### Developer Plan ($29/mes)

**Precio:** $29/mes ó $290/año (ahorro de ~17%)

**Beneficios Basic +:**
- Email support durante business hours
- 1 contacto de soporte
- Response time: 12 horas (general guidance)
- Response time: 24 horas (system impaired)
- Access to AWS Support API

**SLA de Response:**
```
General Guidance:     12 horas
System Imparied:       24 horas
```

**Casos de uso ideal:**
- Desarrollo y testing
- Startups en fase inicial
- Workloads no críticas
- Learning AWS

---

### Business Plan ($100/mes)

**Precio:** $100/mes ó $1,000/año (ahorro de ~17%)

**Beneficios Basic +:**
- 24/7 phone, chat, web support
- Unlimited contacts de soporte
- Response time: <1 hora (production system down)
- Response time: <4 horas (production system impaired)
- Response time: <8 horas (development system impaired)
- Trusted Advisor full checks (incluye todos los checks)
- AWS Support API
- Proactive optimization guidance

**SLA de Response:**
```
Production System Down:     < 1 hora
Production System Impair:  < 4 horas
Development System Impair: < 8 horas
General Guidance:          < 24 horas
```

**Trusted Advisor Full Checks:**
```
Cost Optimization:
  - Unused EC2 instances
  - Unattached EBS volumes
  - Idle load balancers
  - Low RI coverage

Security:
  - Security groups unrestricted
  - IAM access keys rotated
  - MFA on root
  - S3 bucket public access

Fault Tolerance:
  - RDS backup check
  - Multi-AZ deployments
  - Auto Scaling health check

Performance:
  - CloudFront configurations
  - Large EBS volumes
```

**Casos de uso ideal:**
- Production workloads
- E-commerce
- Aplicaciones con SLA de negocio
- Empresas medianas

---

### Enterprise Plan ($15,000/mes)

**Precio:** $15,000/mes ($150,000+ anuales)

**Beneficios Business +:**
- <15 minutos response (business-critical/system down)
- Dedicated Technical Account Manager (TAM)
- Concierge Support Team (para billing y account)
- Well-Architected Framework Reviews (anuales)
- Customized onboarding
- Infrastructure Event Management
- Management tools avanzados

**SLA de Response:**
```
Business-Critical System Down:  < 15 minutos
System Down:                    < 30 minutos
Production System Impaired:     < 1 hora
Development System Impaired:    < 4 horas
General Guidance:               < 12 horas
```

**Casos de uso ideal:**
- Enterprise scale
- Mission-critical applications
- Regulaciones strictas (finance, healthcare)
- Arquitecturas complejas multi-region

## Cómo Seleccionar Plan

### Selección por Budget

```
Budget = $0         → Basic (Free)
Budget < $50/mes    → Considerar Developer
Budget < $200/mes   → Business
Budget > $15,000/mo → Enterprise
```

### Selección por Workload

```
Development/Testing:
  → Basic o Developer

Production (no crítico):
  → Business

Production (crítico con SLAs):
  → Business o Enterprise

Mission-critical:
  → Enterprise
```

### Señales para Upgrade

**Considerar upgrade a Business si:**
- Tienes production workloads
- Necesitas response <1h para system down
- Quieres Trusted Advisor full checks
- Usas AWS Support API para automation

**Considerar upgrade a Enterprise si:**
- Multi-region architecture
- Compliance requirements (HIPAA, PCI, SOC)
- Necesitas TAM dedicado
- Annual Well-Architected Reviews son prioridad

## Comandos Relacionados

```bash
# Crear soporte case (Developer+)
aws support create-case \
  --subject "EC2 instance not responding" \
  --service-code amazon-ec2 \
  --severity-code medium \
  --category-code using-aws-ec2 \
  --communication-body "Description of issue"

# Listar casos
aws support describe-cases \
  --include-resolved-cases false

# Trusted Advisor checks (via CLI)
aws trustedadvisor describe-checks \
  --check-ids [
    "check-123",
    "check-456"
  ]
```

## Tips para Soporte

1. **Usar Business plan mínimo para production** - El costo vale la pena por Trusted Advisor
2. **Documentar antes de abrir caso** - Incluir: resource ARNs, steps to reproduce, error messages
3. **Severity correcto** - No abuse de "critical" si no es crítico
4. **Leverage Support API** - Automatiza casos para issues recurrentes
5. **Trusted Advisor semanalmente** - Revisar recomendaciones de cost optimization
